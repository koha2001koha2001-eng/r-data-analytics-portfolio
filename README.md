# Анализ продуктивности крупного рогатого скота

## 📌 Описание
Анализ данных о коровах: оценка влияния породы и типа рациона на суточный удой и привес.  
Проект демонстрирует полный цикл статистического анализа: от парсинга данных до интерпретации размера эффекта.

## 📊 Данные
- **Источник:** файл `Variant_177.csv` (требует парсинга из-за нестандартной структуры)
- **Переменные:** ID, порода, рацион, лактация, удой (кг), привес (г)
- **Объем после очистки:** зависит от данных (см. вывод скрипта)

## 📈 Основные результаты
- Оценка распределения удоев (асимметрия, эксцесс, Q-Q plot)
- Обнаружение выбросов по привесу (метод IQR)
- Сравнение групп: влияние породы и рациона на удой
- Корреляционный анализ: удой vs привес, лактация vs удой
- Размер эффекта: Cohen's d (рацион), Eta-squared (порода)

## 🖼️ Визуализации
Скрипт создает 8 графиков:
1. Гистограмма + плотность распределения удоя
2. Q-Q Plot для проверки нормальности
3. Boxplot привесов с выделением выбросов
4. Скрипичная диаграмма удоя по породам
5. Распределение рационов по породам
6. Влияние лактации на удой
7. Диаграмма рассеяния: удой vs привес
8. Столбчатая диаграмма с ошибками (SE)
